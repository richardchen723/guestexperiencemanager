# Lightsail Deployment Checklist

Use this checklist to track your deployment progress.

## Pre-Deployment

- [ ] Lightsail instance created (Ubuntu 22.04 LTS)
- [ ] Instance IP address noted
- [ ] SSH access to instance verified
- [ ] Git repository URL ready: `https://github.com/richardchen723/guestexperiencemanager.git`
- [ ] Credentials ready:
  - [ ] OpenAI API key
  - [ ] Hostaway Account ID
  - [ ] Hostaway API Key
  - [ ] Google OAuth credentials (optional)

## Deployment Steps

- [ ] Step 1: Connected to Lightsail instance via SSH
- [ ] Step 2: Initial server setup completed
- [ ] Step 3: PostgreSQL setup completed
- [ ] Step 4: Environment variables configured
- [ ] Step 5: Application deployed
- [ ] Step 6: Systemd service installed and started
- [ ] Step 7: Nginx configured (optional)
- [ ] Step 8: SSL certificate installed (optional)
- [ ] Step 9: Backups configured
- [ ] Step 10: Deployment verified

## Post-Deployment

- [ ] Application accessible
- [ ] Health endpoint responding
- [ ] Database connection working
- [ ] Sync operations tested
- [ ] Logs reviewed
- [ ] Backups tested
